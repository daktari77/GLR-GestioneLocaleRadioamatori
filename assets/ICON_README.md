# Icona portable

Per impostare l'icona dell'EXE portable (PyInstaller):

- Metti qui l'immagine: `assets/gestionale.png` (questa è quella che mi hai passato in chat)
- La build proverà a generare automaticamente `assets/gestionale.ico` (richiede `Pillow`).
- In alternativa, puoi mettere direttamente un `.ico` già pronto in `assets/gestionale.ico`.

La build usa automaticamente `assets/gestionale.ico` se presente.
